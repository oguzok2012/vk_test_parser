# VK Course Test Scraper

Скрипт для автоматизированного прохождения и сбора вопросов из системы тестирования VK.
Работает через перехват API-запросов и использование авторизованной сессии Google Chrome.

## Требования
- Python 3.10+
- Установленный пакетный менеджер **Poetry**
- Установленный браузер Google Chrome
- Скопированный профиль Chrome (для обхода блокировок сессии)

## Установка и настройка окружения

1. Клонируйте репозиторий.
2. Установите зависимости проекта через Poetry:
   ```bash
   poetry install
   ```

3. Создайте файл окружения:
   ```bash
   cp .env.example .env
   ```
4. Откройте файл `.env` и заполните ваши данные (токен авторизации и путь к профилю).

## Настройка профиля Google Chrome (Критически важно!)

Скрипт использует библиотеку `undetected_chromedriver`. Чтобы не проходить авторизацию VK ID (и 2FA) при каждом запуске, мы используем ваш текущий профиль Chrome. 
**СТРОГО ЗАПРЕЩЕНО использовать оригинальную папку профиля**, если у вас параллельно открыт обычный Chrome — это вызовет ошибку блокировки базы данных (`SingletonLock`).

**Инструкция по копированию профиля (macOS):**
1. Закройте ваш основной Google Chrome (Cmd+Q).
2. Скопируйте папку с пользовательскими данными в директорию проекта. Стандартный путь на macOS: 
   `~/Library/Application Support/Google/Chrome`
   
   Выполните в терминале:
   ```bash
   cp -R ~/Library/Application\ Support/Google/Chrome /Абсолютный/Путь/к/проекту/bot_profile
   ```
3. Откройте файл `.env` и укажите абсолютный путь к скопированной папке в переменную `CHROME_PROFILE_DIR`.

*Примечание: Куки сессии VK живут долго, но если скрипт начнет требовать авторизацию при старте, повторите процесс копирования профиля.*

## Запуск скрипта
Используйте команду `run`. Выполняет скрипт внутри окружения Poetry и сразу выходит:
   ```bash
   poetry run python scraper.py
   ```

## Использование
После запуска:
1. Дождитесь открытия браузера.
2. В терминале появится таблица со всеми доступными тестами курса и их статусами (Пройден / Не начат).
3. Введите `ID` нужного теста в консоль.
4. Скрипт начнет скрапинг, перехватывая вопросы из API и предлагая вам выбрать варианты ответов в консоли.
